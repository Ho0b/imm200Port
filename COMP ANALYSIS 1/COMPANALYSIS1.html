UI animation

Explain how each site uses UI animation. Write 2-3 sentences each (total of 6-9 sentences.

        David Milan’s UI is simple, with just a white background and black navigation buttons that change when hovered over. There are columns for images, and it extends down for a while, the main shapes are squares. Though it looks nice, the site does not have many animations from the images themselves, otherwise it's very static.

        By Experience’s website is a modern design where the background is just a solid blue, with a single column, and this single column is the main bulk of visible content.  There are big title sections that say Hello, What and Work then it has layered image effect with view buttons and then their team's main proposal, followed by the footer. What sold me on their site is the layered image effect because when scrolling, the images stays static and the foreground UI changes and moves up. The UI animation on the other hand is very lacking, or rather there arent any at all.

        Finally Studio Bagaz’s website is a collage type design with organized imagery and complementary text. It starts with the main sections which are the information head, the three type of services, design style and footer. The navigational bar is vertical as opposed to the norm, it also has vertical text which makes it hard to read due to the user having to turn their head. Also, the text for their navigation is a little thin, so it is a little hard to read. Otherwise, it is a clean UI. Again, this website does not have much UI animation other than the branding and production section with the three female images changing every second.
